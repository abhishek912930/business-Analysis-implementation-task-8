# Complete Business Analysis Project

## Project Overview
This project analyzes real estate business data to identify the factors influencing property prices and provide business recommendations.

## Dataset
- Total Rows: 1200
- Cleaned Rows: 300
- Features: Area, Bedrooms, Bathrooms, Age, Location, Property Type, Price

## Analysis Techniques
1. Exploratory Data Analysis (EDA)
2. Correlation Analysis
3. Statistical Analysis
4. Comparative Analysis by Property Type and Location

## Visualizations
- Price Distribution
- Area vs Price
- Average Price by Location
- Average Price by Property Type
- Correlation Matrix

## Business Recommendations
- Focus on premium locations with higher average pricing.
- Increase marketing for villas and luxury properties.
- Use property area as a major pricing factor.
- Improve investment in suburban housing projects.
